# CAP — Construção de Algoritmos e Programação

Repositório com códigos desenvolvidos durante a disciplina **CAP (Construção de Algoritmos e Programação)**, cursada no segundo semestre da graduação.

Este repositório reúne exercícios, exemplos de aula e listas propostas ao longo da disciplina, com foco no aprendizado de lógica de programação, algoritmos e fundamentos das linguagens **Python** e **C**.

## Contexto da disciplina
A disciplina de CAP é uma matéria introdutória voltada para:
- desenvolvimento do raciocínio lógico
- construção de algoritmos
- compreensão de estruturas básicas de programação

A matéria foi dividida em duas partes:
- **Python** — introdução à lógica de programação
- **C** — aprofundamento em programação estruturada e conceitos de baixo nível

## Tecnologias utilizadas
- **Python** (primeira parte da disciplina)
- **C** (segunda parte da disciplina)
- Compilador GCC
- Ambiente Linux

## Conteúdos abordados
Os códigos presentes neste repositório cobrem conceitos como:

### Lógica e algoritmos
- Entrada e saída de dados
- Condicionais
- Laços de repetição
- Estruturas de decisão

### Python
- Tipos primitivos
- Funções
- Listas
- Exercícios de lógica

### C
- Variáveis e tipos
- Ponteiros
- Structs
- Vetores e matrizes
- Alocação dinâmica de memória
- Manipulação básica de arquivos
